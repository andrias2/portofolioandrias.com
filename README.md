# Portofolio Saya

Website portofolio pribadi yang menampilkan profil, proyek, dan keahlian teknis saya. Dibuat menggunakan HTML dan CSS murni dengan tema terminal/developer.

🔗 **Live Demo:** https://github.com/andrias2/portofolioandrias.com/

## Tentang

Website ini berisi:
- **Hero** — perkenalan singkat dengan tampilan kartu terminal
- **Tentang Saya** — deskripsi profil dan stack yang digunakan
- **Proyek** — daftar proyek-proyek yang pernah dikerjakan
- **Keahlian** — ringkasan kemampuan teknis
- **Kontak** — cara menghubungi saya

## Teknologi

- HTML5
- CSS3 (tanpa framework, murni custom design)

## Cara Menjalankan Secara Lokal

1. Clone repository ini:
   ```
   git clone https://github.com/USERNAME-ANDA/portfolio-website.git
   ```
2. Masuk ke folder project:
   ```
   cd portfolio-website
   ```
3. Buka file `index.html` langsung di browser.
